Learn HTML & CSS

Step-by-step tutorial to master HTML structure and CSS styling

Step 1: The Basics of HTML (HTML)

HTML stands for HyperText Markup Language, and it is the standard for creating web pages. Every HTML document begins with the <!DOCTYPE> declaration followed by the <html> element:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <!-- Your content goes here -->
  </body>
</html>

Step 2: Headings and Paragraphs (HTML)

Headings help define the structure of your page. HTML offers six levels of headings from <h1> to <h6>. Use these to organize your content:

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-Subheading</h3>

Paragraphs are created using the <p> tag:

<p>This is a paragraph.</p>

Step 3: Emphasis and Strong Text (HTML)

Use <em> for emphasized text, and <strong> for strong importance:

<p>This is <em>emphasized</em> text.</p>
<p>This is <strong>strong</strong> text.</p>

Step 4: Lists (HTML)

Use <ul> for unordered lists and <ol> for ordered lists:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Step 5: Images and Links (HTML)

Use <img> to add images and <a> for links:

<a href="https://example.com">Click here</a>

<img src="image.jpg" alt="Description">

Step 6: Using Div and Span (HTML)

The <div> tag is a block-level container, while <span> is for inline content:

<div class="container">
  <h1>Main Heading</h1>
  <span class="highlight">Inline text</span>
</div>

Step 7: Styling with CSS (CSS)

CSS is used to style your HTML elements. You can use tags like p, .class, and #id to target elements:

/* Styling all paragraphs */
p {
  color: blue;
  font-size: 18px;
}

/* Styling an element with a class */
.container {
  background-color: #f0f0f0;
  padding: 20px;
}

/* Styling an element with an ID */
#highlight {
  color: red;
  font-weight: bold;
}

Step 8: Box Model and Layout (CSS)

CSS uses a box model to determine how elements are displayed. This includes margin, padding, border, and content:

/* Margin: Space outside the element */
.box {
  margin: 20px;
}

/* Padding: Space inside the element */
.box {
  padding: 10px;
}

/* Border: A border around the element */
.box {
  border: 2px solid black;
}

Step 9: Positioning (CSS)

Control the positioning of elements using position, top, left, etc. You can position an element relative to its normal position or absolutely within its container:

/* Position relative to normal flow */
.relative-box {
  position: relative;
  top: 20px;
  left: 30px;
}

/* Position absolutely inside a parent container */
.absolute-box {
  position: absolute;
  top: 0;
  left: 0;
}
